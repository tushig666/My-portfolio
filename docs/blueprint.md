# **App Name**: NeoPulse Tushig

## Core Features:

- Kinetic Particle Hero Stage: An immersive full-screen landing experience using Three.js with floating particles, animated stars, and a moving grid perspective that responds to mouse motion.
- Neuro-Vision Mission Engine: An AI-powered feature that allows visitors to prompt the AI to use its reasoning tool to generate futuristic project visions and custom bio summaries.
- Holographic Tech Stack Galaxy: An interactive, cluster-based tech visualization with 3D tilt effects and neon shadow glows for React, Next.js, and TypeScript skills.
- Glassmorphism Launch Deck: A featured projects gallery utilizing cinematic expansion transitions and animated borders to showcase development work like high-end product launches.
- Axon Persistence Terminal: A database-integrated terminal that captures visitor messages and contact details into a PostgreSQL instance, featuring animated loading states.
- Luminal Timeline Crawler: A scroll-triggered vertical timeline that utilizes GSAP to trace a glowing light path through various career milestones and education nodes.
- Custom Dynamic Adaptive Cursor: A high-frequency cursor implementation that morphs and glows based on interactions with glass cards, magnetic buttons, and holographic UI elements.

## Style Guidelines:

- Primary Color: Electric Cyan (#00DAFF) providing a high-contrast futuristic pulse against the dark aesthetic.
- Background Color: Deep Space Charcoal (#07090A), a heavily desaturated variant of the primary cyan to create depth.
- Accent Color: Aurora Green (#33FFCC), an analogous hue that provides visual separation for highlights and secondary triggers.
- Headline Font: 'Space Grotesk' for a computerized, techy feel across large titles. Body Font: 'Inter' for neutral, machined readability. Code Font: 'Source Code Pro' for micro-interactions.
- Lucide React icons using thin, pixel-perfect stroke weights to maintain the luxury tech startup and minimal cyberpunk feel.
- A single-page cinematic flow utilizing glassmorphism panels, asymmetric grids, and layered z-index depth for a floating OS appearance.
- Lenis for smooth scrolling combined with Framer Motion stagger reveals and magnetic hover effects to ensure a buttery, elite feel.
